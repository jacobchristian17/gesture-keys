    Modify both `run_preview_mode()` in __main__.py and `_detection_loop()` in tray.py with IDENTICAL changes:

    **1. Add import** (top of file):
    ```python
    from gesture_keys.distance import DistanceFilter
    ```

    **2. Create DistanceFilter after config load** (after pipeline component setup):
    ```python
    distance_filter = DistanceFilter(
        min_hand_size=config.min_hand_size,
        enabled=config.distance_enabled,
    )
    ```

    **3. Insert distance check between detect and classify** -- replace the current pattern:
    ```python
    # CURRENT:
    landmarks = detector.detect(frame, timestamp_ms)
    if landmarks:
        raw_gesture = classifier.classify(landmarks)
        gesture = smoother.update(raw_gesture)
    else:
        gesture = smoother.update(None)
    ```
    with:
    ```python
    # NEW:
    landmarks = detector.detect(frame, timestamp_ms)
    if landmarks and not distance_filter.check(landmarks):
        # Hand detected but too far -- reset pipeline state and treat as no hand
        smoother.reset()
        debouncer.reset()
        landmarks = None
    if landmarks:
        raw_gesture = classifier.classify(landmarks)
        gesture = smoother.update(raw_gesture)
    else:
        gesture = smoother.update(None)
    ```

    IMPORTANT: The reset should only happen on the transition from in-range to out-of-range (the DistanceFilter tracks this internally for logging, but the loop needs to avoid resetting every filtered frame). To handle this cleanly, track a `was_in_range` flag in the loop:

    Before the loop: `hand_was_in_range = True`

    In the loop, replace the naive pattern above with:
    ```python
    landmarks = detector.detect(frame, timestamp_ms)
    if landmarks:
        in_range = distance_filter.check(landmarks)
        if not in_range:
            if hand_was_in_range:
                smoother.reset()
                debouncer.reset()
            hand_was_in_range = False
            landmarks = None
        else:
            hand_was_in_range = True
    else:
        hand_was_in_range = True  # No hand = reset tracking for next appearance
    if landmarks:
        raw_gesture = classifier.classify(landmarks)
        gesture = smoother.update(raw_gesture)
    else:
        gesture = smoother.update(None)
    ```

    **4. Add hot-reload update** in the existing config hot-reload block, after the debouncer updates:
    ```python
    distance_filter.enabled = new_config.distance_enabled
    distance_filter.min_hand_size = new_config.min_hand_size
    ```

    **5. In __main__.py only:** Update the config reload log message to include distance info:
    ```python
    logger.info(
        "Config reloaded: %d gestures, delay=%.1fs, cooldown=%.1fs, distance=%s",
        len(new_config.gestures),
        new_config.activation_delay,
        new_config.cooldown_duration,
        "on" if new_config.distance_enabled else "off",
    )
    ```

    Both loops MUST have the same distance gating logic. The only difference is __main__.py has the extra log format and preview rendering.
