    1. `TrayApp` class with `__init__(self, config_path: str)`:
       - Store config_path as absolute path (os.path.abspath)
       - Create `_active = threading.Event()` and call `.set()` (start active)
       - Create `_shutdown = threading.Event()` (start clear)
       - Initialize `_icon = None`, `_detection_thread = None`

    2. `_create_icon_image(self) -> Image.Image`:
       - Create 64x64 RGB Pillow Image with black background
       - Draw a green (#00cc66) filled ellipse [8, 8, 56, 56]
       - Return the image

    3. `_build_menu(self) -> pystray.Menu`:
       - MenuItem: text=lambda showing 'Active'/'Inactive' based on _active.is_set(), action=_on_toggle, checked=lambda returning _active.is_set()
       - Menu.SEPARATOR
       - MenuItem: 'Edit Config', action=_on_edit_config
       - MenuItem: 'Quit', action=_on_quit

    4. `_on_toggle(self, icon, item)`: Toggle _active Event (clear if set, set if clear)

    5. `_on_edit_config(self, icon, item)`: Call `os.startfile(self._config_path)`

    6. `_on_quit(self, icon, item)`:
       - Set _shutdown Event
       - Set _active Event (unblock any _active.wait() in detection loop -- prevents deadlock per research Pitfall 2)
       - Call icon.stop()

    7. `_detection_loop(self)`:
       - Outer loop: `while not self._shutdown.is_set()`
         - `if not self._active.wait(timeout=0.5): continue` (check shutdown while waiting)
         - Load config via load_config(self._config_path)
         - Create CameraCapture, HandDetector, GestureClassifier, GestureSmoother, GestureDebouncer, KeystrokeSender, ConfigWatcher
         - Pre-parse key_mappings via _parse_key_mappings (import from __main__ or duplicate the helper)
         - Inner loop: `while self._active.is_set() and not self._shutdown.is_set()`
           - Same detection pipeline logic as current main() but without preview rendering
           - Include config hot-reload via ConfigWatcher
         - After inner loop: camera.stop(), detector.close() (release resources when inactive or shutting down)

    8. `_start_detection(self)`:
       - Create daemon thread targeting _detection_loop, start it

    9. `run(self)`:
       - Create pystray.Icon('gesture-keys', icon=_create_icon_image(), title='Gesture Keys', menu=_build_menu())
       - Store as self._icon
       - Call self._icon.run(setup=lambda icon: self._start_detection())

    **IMPORTANT:** Move `_parse_key_mappings` from `__main__.py` into a shared location. Best approach: keep it in `__main__.py` and import it in `tray.py`, OR duplicate the small helper in tray.py. Since __main__.py is not a clean import target, duplicate the 10-line helper in tray.py.
