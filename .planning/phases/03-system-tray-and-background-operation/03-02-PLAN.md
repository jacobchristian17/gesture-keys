       ```python
       def hide_console_window():
           """Hide the console window on Windows (tray mode only)."""
           if sys.platform == 'win32':
               import ctypes
               hwnd = ctypes.windll.kernel32.GetConsoleWindow()
               if hwnd:
                   ctypes.windll.user32.ShowWindow(hwnd, 0)  # SW_HIDE
       ```
       Add `import sys` to the top imports.

    2. Add a `run_tray_mode(args)` function:
       ```python
       def run_tray_mode(args):
           """Run in system tray mode (default, no preview)."""
           hide_console_window()

           app = TrayApp(config_path=args.config)
           app.run()
       ```

    3. Rename the current `main()` body to `run_preview_mode(args)`:
