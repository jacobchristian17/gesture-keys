  <files>gesture_keys/config.py, gesture_keys/debounce.py, tests/test_config.py, tests/test_debounce.py</files>
  <behavior>
    - AppConfig() with no args produces gesture_cooldowns={}
    - load_config() extracts per-gesture cooldowns from gesture entries that have a "cooldown" key
    - load_config() with gestures.pinch.cooldown: 0.6 returns gesture_cooldowns={"pinch": 0.6}
    - load_config() ignores gestures without a cooldown key (they use global)
    - GestureDebouncer with gesture_cooldowns={"fist": 0.5} uses 0.5s cooldown when FIST fires, global default for others
    - GestureDebouncer with no gesture_cooldowns uses global cooldown_duration for all gestures
    - Debouncer _handle_cooldown uses the per-fire cooldown duration (not the global) for elapsed check
  </behavior>
  <action>
1. Add gesture_cooldowns field to AppConfig in config.py:
   ```python
   gesture_cooldowns: dict[str, float] = field(default_factory=dict)
   ```

2. Add helper function to extract per-gesture cooldowns in config.py:
   ```python
   def _extract_gesture_cooldowns(gestures: dict) -> dict[str, float]:
       cooldowns = {}
       for name, settings in gestures.items():
           if isinstance(settings, dict) and "cooldown" in settings:
               cooldowns[name] = float(settings["cooldown"])
       return cooldowns
   ```

3. Call in load_config() and pass to AppConfig:
   ```python
   gesture_cooldowns=_extract_gesture_cooldowns(gestures),
   ```

4. Update GestureDebouncer.__init__ in debounce.py to accept gesture_cooldowns:
   ```python
   def __init__(self, activation_delay: float = 0.15, cooldown_duration: float = 0.3,
                gesture_cooldowns: dict[str, float] | None = None) -> None:
       self._cooldown_duration = cooldown_duration
       self._gesture_cooldowns = gesture_cooldowns or {}
       self._cooldown_duration_active = cooldown_duration  # per-fire cooldown value
   ```
   Use optional kwarg with None default for backward compatibility (per Phase 08 pattern).

5. Update _handle_fired() in debounce.py to set per-fire cooldown:
   ```python
   def _handle_fired(self, gesture, timestamp):
       self._state = DebounceState.COOLDOWN
       self._cooldown_start = timestamp
       fired_gesture = self._activating_gesture
       self._cooldown_duration_active = self._gesture_cooldowns.get(
           fired_gesture.value, self._cooldown_duration
       )
       self._cooldown_gesture = fired_gesture
       self._activating_gesture = None
       logger.debug("FIRED -> COOLDOWN")
       return None
   ```

6. Update _handle_cooldown() in debounce.py to use _cooldown_duration_active:
   Change the elapsed check from:
   `timestamp - self._cooldown_start >= self._cooldown_duration`
   To:
   `timestamp - self._cooldown_start >= self._cooldown_duration_active`

7. Update reset() in debounce.py to reset _cooldown_duration_active:
   ```python
   self._cooldown_duration_active = self._cooldown_duration
   ```

8. Add tests in tests/test_config.py:
   - Test AppConfig default gesture_cooldowns is {}
   - Test load_config extracts per-gesture cooldown from YAML
   - Test load_config ignores gestures without cooldown key
   - Test load_config handles mixed (some with, some without cooldown)

9. Add tests in tests/test_debounce.py:
   - Test debouncer with gesture_cooldowns uses per-gesture cooldown duration for that gesture
   - Test debouncer without gesture_cooldowns uses global cooldown_duration
   - Test debouncer with gesture_cooldowns falls back to global for gestures not in dict
   - Test debouncer per-gesture cooldown: fire pinch (0.6s cooldown) -> release at 0.5s -> still in cooldown; release at 0.7s -> idle
