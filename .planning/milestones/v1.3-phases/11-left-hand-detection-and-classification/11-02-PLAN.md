    **Classifier analysis:** The GestureClassifier uses:
    - `_is_finger_extended`: `tip.y < pip.y and pip.y < mcp.y` — purely vertical, hand-agnostic.
    - `_is_thumb_extended`: `abs(tip.x - wrist.x) > abs(ip.x - wrist.x)` — uses `abs()`, hand-agnostic.
    - `_is_pinch`: Euclidean distance between thumb tip and index tip — hand-agnostic.

    The classifier is already hand-agnostic. The key question is whether mirrored landmark positions from MediaPipe for a left hand produce the same geometric relationships. Since MediaPipe normalizes landmarks per-hand, the answer is likely yes.

    **Create left-hand fixtures (conftest.py):**
    For left-hand landmarks, the thumb is on the RIGHT side of the hand (higher x values) instead of the left side. Create mirrored fixtures by reflecting thumb x-positions around the wrist x-center:

    - `mock_landmarks_left_fist`: Mirror fist. Thumb tip at x=0.52 (close to wrist = curled, reflected).
    - `mock_landmarks_left_thumbs_up`: Mirror thumbs_up. Thumb tip at x=0.8 (far from wrist = extended, reflected).
    - `mock_landmarks_left_peace`: Mirror peace. Thumb curled reflected.
    - `mock_landmarks_left_pointing`: Mirror pointing. Thumb curled reflected.
    - `mock_landmarks_left_pinch`: Mirror pinch. Thumb tip and index tip remain close together but thumb on right side. Thumb IP at x=0.62, thumb tip at x=0.65 (further from wrist x=0.5 = extended).
    - `mock_landmarks_left_scout`: Mirror scout. Thumb curled reflected.

The thumb is "extended" when its tip is further from the wrist than its IP joint, regardless of direction. Verify this by running tests first (RED phase should actually PASS if classifier is truly agnostic — that's the goal). If any test fails, adjust the classifier to handle left-hand geometry.

    ```python
    # Hand switch detection: reset pipeline on hand change
    if handedness is not None and prev_handedness is not None and handedness != prev_handedness:
        # Per CONTEXT.md: instant switch, release holds, reset pipeline
        hold_active = False
        sender.release_all()
        smoother.reset()
        debouncer.reset()
    ```
