1. Add new fields to AppConfig dataclass:
   - `left_gestures: dict[str, dict[str, Any]]` (default empty dict) — raw left-hand gesture overrides from YAML
   - `left_swipe_mappings: dict[str, str]` (default empty dict) — left-hand swipe key overrides
   - `left_gesture_cooldowns: dict[str, float]` (default empty dict)
   - `left_gesture_modes: dict[str, str]` (default empty dict)

2. Update `load_config()` to parse new YAML sections:
   - Parse optional `left_gestures:` top-level section (same structure as `gestures:` — each entry has key, threshold, cooldown, mode). Only entries the user wants to override need to be present.
   - Parse optional `left_swipe:` top-level section (same structure as `swipe:` direction entries — `swipe_left: {key: ...}`).
   - Extract left_gesture_cooldowns via `_extract_gesture_cooldowns(left_gestures)`.
   - Extract left_gesture_modes via `_extract_gesture_modes(left_gestures)`.

3. Create two resolution functions (module-level, not methods):
   - `resolve_hand_gestures(handedness: str, config: AppConfig) -> dict`: Returns the gesture dict for the active hand. When handedness is "Right" (MediaPipe format) or left_gestures is empty, return config.gestures. When "Left" and left_gestures has entries, deep-merge: start with copy of config.gestures, then for each gesture in left_gestures, merge that gesture's settings dict over the right-hand entry (so left overrides only what it specifies, threshold/cooldown/mode fall back to right-hand values).
   - `resolve_hand_swipe_mappings(handedness: str, config: AppConfig) -> dict[str, str]`: When "Right" or left_swipe_mappings is empty, return config.swipe_mappings. When "Left" and left_swipe_mappings has entries, return left_swipe_mappings (full replacement, not merged — swipe directions are atomic).

YAML structure decision: Use `left_gestures:` as a top-level section mirroring the `gestures:` structure. Use `left_swipe:` mirroring the `swipe:` direction entries. This keeps the config clean and predictable — users copy a gesture block, change the key, paste under `left_gestures:`.

Example config.yaml with left-hand overrides:
```yaml
gestures:
  open_palm:
    key: win+tab
    threshold: 0.7

left_gestures:
  open_palm:
    key: ctrl+w    # Only override the key; threshold/mode inherited from gestures.open_palm
```
