    - test_different_gesture_during_cooldown_eventually_fires:
      FIST fires, enters cooldown. PEACE during cooldown -> ACTIVATING. Hold PEACE for activation_delay -> fires PEACE.

    - test_same_gesture_during_cooldown_stays_blocked:
      FIST fires, enters cooldown. FIST at t=0.7 -> still COOLDOWN, returns None. (Existing test validates post-elapsed; this validates mid-cooldown.)

    - test_same_gesture_after_cooldown_elapsed_stays_blocked:
      FIST fires, enters cooldown. Cooldown elapses. FIST still held -> stays COOLDOWN. Must release to None first.

    - test_rapid_switch_during_cooldown_fires_final_gesture:
      FIST fires, cooldown. PEACE at t=0.7 -> ACTIVATING(PEACE). POINTING at t=0.8 -> resets to ACTIVATING(POINTING). Hold POINTING -> fires POINTING.

    - test_cooldown_gesture_cleared_on_transition_to_activating:
      After COOLDOWN->ACTIVATING transition, _cooldown_gesture should be None (no stale state).

    - test_cooldown_gesture_cleared_on_transition_to_idle:
      After COOLDOWN->IDLE (release after elapsed), _cooldown_gesture should be None.

    - test_reset_clears_cooldown_gesture:
      Set up cooldown state with _cooldown_gesture set, call reset(), verify _cooldown_gesture is None.

    - test_different_gesture_after_cooldown_elapsed_starts_activating:
