<task type="auto" tdd="true">
  <name>Task 2: Gesture classifier and majority-vote smoother</name>
  <files>
    gesture_keys/classifier.py,
    gesture_keys/smoother.py,
    tests/test_classifier.py,
    tests/test_smoother.py

  <behavior>
    Classifier:
    - classify(landmarks) returns Gesture enum value for each of the 6 gestures
    - open_palm: all 4 fingers extended + thumb extended -> OPEN_PALM
    - fist: all 4 fingers curled + thumb curled -> FIST
    - thumbs_up: thumb extended + all 4 fingers curled -> THUMBS_UP
    - peace: index + middle extended, ring + pinky curled -> PEACE
    - pointing: index extended, middle + ring + pinky curled -> POINTING
    - pinch: thumb tip close to index tip (euclidean distance < threshold) -> PINCH
    - Priority order: PINCH > FIST > THUMBS_UP > POINTING > PEACE > OPEN_PALM > None
    - Ambiguous poses return None
    - Accepts optional threshold parameter per gesture from config

    Smoother:
    - GestureSmoother(window_size=3) creates buffer
    - update(gesture) returns None until buffer is full (first window_size-1 calls)
    - After buffer full, returns majority vote of last window_size values
    - Tie-breaking: if no majority, returns None (conservative)
    - Handles None as a valid gesture value in the buffer
  </behavior>
  <action>
    1. Create `gesture_keys/classifier.py`:
       - Define `Gesture` enum: OPEN_PALM, FIST, THUMBS_UP, PEACE, POINTING, PINCH (and None for no gesture -- use None, not an enum member)
       - Define `GestureClassifier` class:
         - Constructor takes gesture thresholds dict from config (per-gesture thresholds, default 0.7)
         - `classify(landmarks)` method that takes a list of landmark objects (with .x, .y, .z)
         - Internal helpers: `_is_finger_extended(landmarks, tip_idx, pip_idx)` -- tip.y < pip.y means extended
         - `_is_thumb_extended(landmarks)` -- compare thumb tip x-distance from wrist vs thumb IP x-distance from wrist (see RESEARCH.md Pattern 4, Pitfall 6)
         - `_is_pinch(landmarks, threshold)` -- euclidean distance between landmark[4] and landmark[8] < threshold
         - Classification priority per CONTEXT.md: check PINCH first, then FIST, THUMBS_UP, POINTING, PEACE, OPEN_PALM, else None
       - Use landmark indices from RESEARCH.md: FINGER_TIPS = [8, 12, 16, 20], FINGER_PIPS = [6, 10, 14, 18], THUMB_TIP = 4, THUMB_IP = 3, WRIST = 0, INDEX_TIP = 8

    2. Create `gesture_keys/smoother.py`:
       - `GestureSmoother` class with `deque(maxlen=window_size)` buffer
       - `update(gesture)` appends to buffer, returns majority vote via `collections.Counter` when buffer full
       - If Counter most_common count <= window_size/2 (no clear majority), return None
       - `reset()` method to clear the buffer

    3. Create `tests/test_classifier.py`:
       - Test each gesture with the mock landmarks from conftest.py
       - Test priority ordering (e.g., when both pinch and pointing could match, pinch wins)
       - Test ambiguous pose returns None
       - Test with custom thresholds

    4. Create `tests/test_smoother.py`:
       - Test buffer fills before reporting
       - Test majority vote with clear winner
       - Test tie returns None
       - Test None is a valid value in buffer
       - Test reset clears state

    Run full test suite.
